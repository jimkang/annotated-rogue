• Annotated Rogue source

    This is an attempt to annotate the source for the version of Rogue that was distributed with BSD in 1987.

    It's a reimplementation the original Rogue by Tim Stoehr and others that has an unambiguous BSD license and is fairly close to the original, according to John Harris's book. It's called "rogue-clone: Version III" in the source. It's buildable and playable on the OSes of 2016, including OS X 10.10.

    You can get the source for Rogue 3.6, a version of the original Rogue from 1981, at the Roguelike Restoration Project. It's going to be hard to compile, but it's still readable. I personally find Rogue Clone's source to be better organized, which probably follows from time Stoehr being able to start from scratch knowing exactly what to build.

    The templates for these web pages were generated by docco (in particular, this fork, which is very aggressive about deciding when to make a new section).
